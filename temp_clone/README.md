# Creative Director Portfolio

A professional portfolio built with Next.js, Tailwind CSS, and Framer Motion. Deployed on Vercel for free.

## Features

- Dark/Light mode toggle
- Smooth scroll with Lenis
- Scroll-triggered animations with Framer Motion
- Decap CMS for content management
- Responsive design
- Static export for easy hosting

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000)

## Deployment

1. Push to GitHub
2. Connect repo to Vercel
3. Auto-deploys on every push

## Content Management

Visit `/admin/` on your deployed site to access Decap CMS. Configure Git Gateway in Netlify or use GitHub OAuth.

## Customization

- Replace "YOUR NAME" in Navigation and Footer components
- Update contact email in Contact section
- Add your own project images to `public/images/`
- Update social links in Contact and Footer
